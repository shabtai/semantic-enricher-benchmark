Lift (50% → 92%)

The same agent answers the same questions against the same raw data files in both arms. The only experimental variable is whether the agent has access to enrichment guidance for the dataset. Each error in the unenriched arm is attributable to the agent operating without that guidance: it produces an articulate, plausible answer that does not match the scientific meaning of the question. With the enrichment in scope, the agent reaches the correct interpretation and either answers correctly or appropriately declines.

Where the lift comes from. The benefit of enrichment is largest where the agent's prior knowledge is weakest. On a post-cutoff chromatin dataset (the model has not seen it during training), the unenriched score was 3/8 and the enriched score was 8/8. On MIMIC-III (heavily represented in the public literature), prior knowledge already supplies most of what is needed and the lift is a single question.

The MIMIC result behaves as a control: where the agent already has strong prior knowledge of a dataset, enrichment provides only marginal additional benefit. The bulk of the lift in this benchmark comes from datasets where the agent has no such prior.

Failure mechanism

Errors in the unenriched arm are not produced by reasoning failures or numeric mistakes. The wrong answers are articulate, internally consistent, and accompanied by reasonable-looking caveats. The failure mode is that the agent operates on a misinterpretation of what the data represents:

  1. Plausible-but-wrong path of least resistance. When the agent does not know that a table is already an aggregate of another, the natural choice is to join through the lower-level table and double-count. When it does not know that three files contain the same calls at different resolutions, the natural choice is to sum.
  2. Filenames carry intent that requires guidance to disambiguate. Names containing tokens such as "random", "polysim", or a reference-genome build do encode the answer — but the agent treats the file as data without questioning whether it represents study output, a control, a simulation, or a reference annotation.

In every failure observed, the underlying issue is that the agent is missing the dataset-specific scientific context that the enrichment supplies.

Method

  • Identical agent and prompt. Both arms run the same agent, the same prompt template, and the same answering workflow over the same raw data files. The only experimental variable is whether the dataset's enrichment is in scope.
  • Scoring. An LLM judge reads the agent's answer against the question and the expected scientific interpretation, returning PASS or FAIL with a written rationale. PASS requires either a correct numerical answer with appropriate caveats, or an appropriate refusal grounded in the scientific reason the question cannot be answered as posed. A correct number presented without acknowledgement of the relevant scientific qualification is scored as FAIL.
  • Unenriched ≠ uninformed. The unenriched arm has full access to the raw data and unrestricted reasoning capacity. The comparison is therefore between training-data prior alone and training-data prior augmented by dataset-specific structured guidance, not between no knowledge and full knowledge.
  • No priming. The questions are issued without hints about the expected interpretation or the relevant scientific qualifications.
